# Movision Version 1 

Developed from scratch by Ozan Gokdemir for the Android Developer Nanodegree, Popular Movies Stage 1 project requirement.


# Features
- Allows for making queries to the TMDB API to view movies ordered by popularity, top ratings, upcoming and now playing.
- Dynamic and adaptable UI design. 
- Optimized network operations. 
- Fast and concise code made easier to read by incorporating four powerful 3rd party libraries: Butterknife, GSON, Parceler, Picasso.


# Important note for the testers and clients!

The app uses [TMDB Movie API] (https://www.themoviedb.org/documentation/api) 

*Please insert your own api key into res/values/strings.xml, api_key field.*



# Libraries
- [Picasso](http://square.github.io/picasso/)
- [Butterknife](http://jakewharton.github.io/butterknife/)
- [Parceler] (http://parceler.org/)
- [GSON] (https://github.com/google/gson)

# Testing and Debugging
 
*The app is tested on Nexus 5x running Oreo(API 26). 
*The app is targeted for Android devices running API 21 and higher.
